Hypothesis Testing Framework

Hypothesis testing is a statistical method used to make decisions about a population based on sample data. In dental research it helps determine whether a treatment, intervention, or clinical observation has a real effect or whether the observed result occurred by chance.

Why use hypothesis testing?

For example, a researcher may ask whether a new fluoride treatment reduces dental caries more effectively than standard treatment. Hypothesis testing provides a structured way to evaluate such questions using sample data.

The Hypothesis Testing Process

Hypothesis testing typically follows this sequence of steps:

  1. State the hypotheses

    Null hypothesis (H0): the default assumption that there is no effect, difference, or relationship.

    Alternative hypothesis (H1 or Ha): the statement that a real effect or difference exists (what the researcher wants to show).

    Example: Testing whether a new toothpaste reduces plaque more than the standard toothpaste. H0: no difference. H1: new toothpaste reduces plaque more.

  2. Select the significance level (α)
    α is the probability of making a Type I error (false positive). Common choices: 0.05 (5%), 0.01, or 0.001 for stricter evidence.
  3. Calculate the test statistic
    Compute a statistic from the sample (e.g., t value, χ², F) appropriate for the chosen test (t-test, chi-square, ANOVA, regression).
  4. Determine the p-value
    The p-value is the probability of observing the data (or more extreme) assuming H0 is true. If p ≤ α, reject H0; otherwise fail to reject H0.
  5. Draw the conclusion
    Decide whether the evidence is strong enough to reject H0. Remember, failing to reject H0 does not prove H0 is true; it only indicates insufficient evidence.

Test statistic examples

  • t-test – t value (compare means)
  • Chi-square – χ² value (categorical associations)
  • ANOVA – F value (compare >2 group means)
  • Regression – t or F (effect of predictors)

Interpreting p-values

Compare p-value to α:

  • If p ≤ α → reject H0 (evidence supports H1).
  • If p > α → fail to reject H0 (insufficient evidence).

Types of errors

Type I error (α): false positive (rejecting a true H0). Example: concluding a fluoride varnish prevents caries when it does not.

Type II error (β): false negative (failing to reject a false H0). Example: missing a real benefit of a periodontal therapy due to small sample size or high variability.

Statistical power

Power = 1 − β is the probability of detecting a true effect. Common targets: 0.80 (80%) or 0.90 (90%). Low power increases the risk of Type II errors.

Practical examples

Example 1: A study comparing two periodontal treatments reports p = 0.03 (α = 0.05). Since 0.03 < 0.05, the result is statistically significant and H0 is rejected.

Example 2: A whitening gel study finds p = 0.20 (α = 0.05). Since 0.20 > 0.05, the result is not statistically significant and there is insufficient evidence to claim a treatment effect.

Summary table

Concept Meaning

Significance level (α) Probability of Type I error (commonly 0.05)
p-value Probability results occurred by chance under H0

Notes: Choose appropriate tests, ensure adequate sample size to achieve desired power, and report both effect sizes and p-values for transparent interpretation.
